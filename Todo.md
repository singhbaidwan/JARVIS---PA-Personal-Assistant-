# 🧠 JARVIS – Personal AI Assistant (Mac)

## 🧩 Phase 0: Core Vision

A **local-first, intelligent, adaptive OS-level assistant** that:

* Understands user behavior
* Automates workflows
* Optimizes system usage
* Acts proactively (not just reactive commands)
